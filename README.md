# Reien
軽量で小規模インスタンス向けのActivityPub実装「reien」

## 技術スタック
- 言語/Typescript
- ランタイム/Bun
- バックエンドフレームワーク/hono.js
- フロントエンドフレームワーク/Vue.js
- ActivityPub/fedify
- データベース/sqlite
- メール/nodemailer
- 画像処理/fluent-ffmpeg
